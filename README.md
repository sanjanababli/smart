# Smart Business Inventory Management System

Production-ready MERN project scaffold with a separated frontend and backend.

## Structure

- `frontend`: React + Vite client
- `backend`: Node.js + Express API with MongoDB (Mongoose)

## Quick Start

1. Backend setup
   - `cd backend`
   - `npm install`
   - `Copy-Item .env.example .env`
   - `npm run dev`
2. Frontend setup
   - `cd frontend`
   - `npm install`
   - `Copy-Item .env.example .env`
   - `npm run dev`
